Son chapitre 1er, qui concerne l'ensemble du territoire national, permet de ne pas appliquer les délais majorés prévus par le code de l'urbanisme dans plusieurs cas de figure - lorsque le projet est situé dans le périmètre d’un site patrimonial remarquable, aux abords d’un monument historique, ou lorsqu’il est soumis à la participation du public.
